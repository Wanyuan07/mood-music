# Mood Music

Mood Music is a web application that recommends music based on the user’s mood using Spotify playlists and the Spotify Web API.

## Features
- Mood-based music recommendations
- Spotify playlist integration
- Dynamic song rendering
- Search functionality
- Responsive design
- Interactive UI with hover effects
- Randomized recommendations

## Tech Stack
- HTML
- CSS
- JavaScript
- Spotify Web API

## Setup
1. Get a Spotify access token
2. Replace `YOUR_TOKEN` in `home.js`
3. Open `index.html`
